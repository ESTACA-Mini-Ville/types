# types

Repository of IDL (Interface Definition Language) message definitions used by the ESTACA PIRATE project.

This repository contains a small set of DDS-compatible IDL files. The files are organized into folders by logical message package (`geometry_msgs` and `std_msgs`). These IDL files are intended to be built and installed into a DDS workspace/
